YPbPr2RGB R Documentation

Convert YCbCr coordinates to RGB

Description

YPbPr2RGB Converts YCbCr coordinates to RGB.

Usage

YPbPr2RGB(YPbPrmatrix)

Arguments

YPbPrmatrix

YPbPr coordinates

Value

RGB coordinates
