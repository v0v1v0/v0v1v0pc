Usage

uai(eto, pr, data_names = NULL, time.scale = YEAR, na.rm = FALSE)

Arguments

pr

data_names

names of each period of time
time.scale

month, season or year
na.rm

logical. Should missing values (including NaN) be removed?

Value

index value
