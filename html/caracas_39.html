Description

Difference matrix

Usage

diff_mat(N, l = "-1", d = 1)

Arguments

N

Number of rows (and columns)
l

Value / symbol below main diagonal
d

Value / symbol on main diagonal

Examples

if (has_sympy()){
  Dm <- diff_mat(4)
  Dm
  y <- vector_sym(4, "y")
  Dm %*% y
}
