c_s_h

Description

Smooth function.

Usage

c_s_h(t, h)

Arguments

t

A value, the difference between any two subjects.
h

The value of h falls into (n^(-1/2), n^(-1/5)).

Details

Smooth function.

Value

s_h

The value of smooth function.
