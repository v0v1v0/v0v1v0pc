nice_DAG_plot R Documentation

nice_DAG_plot

Description

DAG visualization

Usage

nice_DAG_plot(
  my_DAG,
  print_direct = TRUE,
  node_size = NULL,
  CPDAG = TRUE,
  node_colours = "#fdae61",
  directed = TRUE
)

Arguments

my_DAG

DAG
print_direct

print DAG if TRUE
node_size

node size vector
CPDAG

if TRUE, then plot CPDAG instead of DAG
node_colours

node colours
directed

TRUE if nodes should be directed

Value

A plot of the DAG of class c("gg", "ggplot").
