c3_line R Documentation

Line Plot

Description

Add lines to a C3 plot

Usage

c3_line(c3, type, stacked = FALSE, connectNull = FALSE,
  step_type = NULL)

Arguments

c3

c3 htmlwidget object
type

character type of line plot. Must be one of:

  • line

  • spline

  • step

  • area

  • area-step

stacked

boolean
connectNull

boolean connect null (missing) data points
step_type

character, one of:

  • step

  • step-after

  • step-before

Value

c3

Examples

data.frame(a=c(1,2,3,2),b=c(2,3,1,5)) %>%
  c3() %>%
  c3_line('spline')
