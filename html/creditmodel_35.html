date_cut R Documentation

Date Time Cut Point

Description

date_cut is a small function to get date point.

Usage

date_cut(dat_time, pct = 0.7, g = 100)

Arguments

dat_time

time vectors.
pct

the percent of cutting. Default: 0.7.
g

Number of cuts.

Value

A Date.

Examples

date_cut(dat_time = lendingclub$issue_d, pct = 0.8)
#"2018-08-01"
