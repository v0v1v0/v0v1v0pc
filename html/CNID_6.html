birth_year R Documentation

Get year of birth information from ID number

Description

Get year of birth information from ID number.

Usage

birth_year(id)

Arguments

id

A vector of ID numbers.

Value

Year of birth vector obtained from ID numbers.

Examples

id = c(
"652801197305161555",
"130206202202291545", 
"110101841125178"
)
birth_year(id)
