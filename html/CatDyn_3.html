catdyn R Documentation

Class Attribute of Numerically Fit CatDyn Model Objects

Description

To be used by CatDynPred() to create model results.

Usage

catdyn(x, ...)

Arguments

x

A list object coming from the optimization wrapper CatDynFit().

...

Not used.

Value

A class attribute.
