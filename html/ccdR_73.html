get_hazard_by_dtxsid_batch_old R Documentation

Get hazard data by DTXSID batch

Description

Get hazard data by DTXSID batch

Usage

get_hazard_by_dtxsid_batch_old(
  DTXSID = NULL,
  API_key = NULL,
  rate_limit = 0L,
  verbose = FALSE
)

Arguments

DTXSID

A list of chemical identifier DTXSIDs
API_key

The user-specific API key
rate_limit

Number of seconds to wait between each request
verbose

A logical indicating if some “progress report” should be given.

Value

A named list of data.frames containing chemical (human and eco) hazard data for each input chemical.
