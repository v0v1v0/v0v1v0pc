CMYK2CMY R Documentation

Convert CMYK coordinates to CMY

Description

CMYK2CMY Converts CMYK coordinates to CMY.

Usage

CMYK2CMY(CMYKmatrix)

Arguments

CMYKmatrix

CMYK coordinates

Value

CMY coordinates

Author(s)

Jose Gama

Source

References

Logicol S.r.l., 2014 EasyRGB color search engine http://www.easyrgb.com/

Examples

CMYK2CMY(c(.342, .768, .683, .378))
