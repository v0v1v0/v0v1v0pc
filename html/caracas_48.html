Expand expression

Description

Expand expression

Usage

expand(x, ...)

Arguments

x

A caracas_symbol
...

Pass on to SymPy's expand, e.g. force = TRUE

Examples

if (has_sympy()) {
  def_sym(x)
  y <- log(exp(x))
  simplify(y)
  expand(simplify(y))
  expand(simplify(y), force = TRUE)
  expand_log(simplify(y))
}
