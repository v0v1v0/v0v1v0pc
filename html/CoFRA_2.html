Description

This function filter a data frame on column named "pro"

Usage

filterData(dfPro, filter)

Arguments

dfPro

data frame with iBAQ values
filter

character vector with items to remove

Examples

library(CoFRA)
data(iBAQ)
iBAQ2=filterData(iBAQ,">CON") # filter headers starting with >CON
