birth_month R Documentation

Get month of birth information from ID number

Description

Get month of birth information from ID number.

Usage

birth_month(id)

Arguments

id

A vector of ID numbers.

Value

Month of birth vector obtained from ID numbers.

Examples

id = c(
"652801197305161555",
"130206202202291545", 
"110101841125178"
)
birth_month(id)
