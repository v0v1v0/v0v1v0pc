pseudobullseye R Documentation

Pseudo bullseye plot

Description

Pseudo bullseye plot

Usage

pseudobullseye(
  x,
  lim = range(x, na.rm = TRUE),
  legend = FALSE,
  text = TRUE,
  reverse = FALSE,
  center = TRUE,
  cex = 1,
  legend.width = 1
)

Arguments

x

3D array
lim

limits of x values
legend

boolean, add legend?
text

boolean, should text legend be added?
reverse

boolean, reverse colors?
center

boolean, should input x be centered before plotting
cex

cex for text legend
legend.width

Width in characters of the legend strip.

Value

plots

Examples

data(cmrsim)
pseudobullseye(maxresp_sim)
