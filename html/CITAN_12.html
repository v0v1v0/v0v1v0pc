lbsDisconnect R Documentation

Disconnect from a Local Bibliometric Storage

Description

Disconnects from a Local Bibliometric Storage.

Usage

lbsDisconnect(conn)

Arguments

conn

database connection object, see lbsConnect.

See Also

lbsConnect

Examples

## Not run: 
conn <- lbsConnect("Bibliometrics.db");
## ...
lbsDisconnect(conn);
## End(Not run)
