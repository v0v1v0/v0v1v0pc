expcub00 R Documentation

Expectation of CUB distributions

Description

Compute the expectation of a CUB model without covariates.

Usage

expcub00(m,pai,csi)

Arguments

m

Number of ordinal categories
pai

Uncertainty parameter
csi

Feeling parameter
