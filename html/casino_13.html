Slots R6 Class

Description

Slots R6 Class

Usage

Slots

Format

An object of class R6ClassGenerator of length 24.

Examples

set.seed(101315)
setup()

# start the slot machine
x <- Slots$new(who = "Player 1", bet = 10)

# play 1 game
x$play()

# play >1 game at a time
x$play(spins = 3)

# clean-up
delete()
