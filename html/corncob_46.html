soil_phylo_taxa R Documentation

Soil data, taxa table as data frame.

Description

A data frame made from a soil 'phyloseq' object with only taxonomy data.

Usage

soil_phylo_taxa

Format

A phyloseq-class experiment-level object with an OTU table.

tax_table

taxonomy table
