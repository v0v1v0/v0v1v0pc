path_rect R Documentation

Rectangle Path

Description

Calculate the path of a rectangle in a grid.

Usage

path_rect(yx1, yx2)

Arguments

yx1

upper-left (row, col) coordinate
yx2

lower-right (row, col) coordinate

Value

Nx2 matrix of (row, column) coordinates

See Also

Other path-fitting functions: path_arc(), path_bezier(), path_circle(), path_ellipse(), path_fill(), path_fn(), path_intersection(), path_lerp(), path_ray(), path_shape()

Examples

path_rect(c(5,5), c(9,9))
