circle R Documentation

Draw a circle

Description

circle draws a circle in an existing plot.

Usage

circle(radius,origin)

Arguments

radius

the radius of the circle

origin

the origin of the circle

Value

NULL

Author(s)

Jan Graffelman jan.graffelman@upc.edu

Examples

   set.seed(123)
   X <- matrix(rnorm(20),ncol=2)
   plot(X[,1],X[,2])
   circle(1,c(0,0))
