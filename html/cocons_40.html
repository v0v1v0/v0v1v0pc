plot,coco,missing-method R Documentation

Plot Method for Coco Class

Description

This method plots objects of class coco.

Usage

## S4 method for signature 'coco,missing'
plot(x, y, ..., type = NULL, index = NULL, factr = 0.1, plot.control = NULL)

Arguments

x

An object of class coco.
y

Not used.
...

Additional arguments passed to the plot function. when type "ellipse" , delta of nearest.dist must be specified.
type

The type of plot. NULL or "ellipse" for drawing ellipse of the convolution kernels.
index

For plotting local correlation plots.
factr

Factor rate for size of ellipses.
plot.control

Additional plot control parameters.

Value

A plot is created.
