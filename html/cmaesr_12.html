stopOnNoEffectCoord R Documentation

Stopping condition: standard deviation in coordinates.

Description

Stop if addition of 0.2 * standard deviations in any coordinate does not change mean value.

Usage

stopOnNoEffectCoord()

Value

[cma_stopping_condition]
