cb_load_cohort R Documentation

Get cohort information

Description

Get all the details about a cohort including applied query.

Usage

cb_load_cohort(cohort_id, cb_version = "v2")

Arguments

cohort_id

Cohort id (Required)
cb_version

cohort browser version (Optional) [ "v1" | "v2" ]

Value

A cohort object.

See Also

cb_create_cohort for creating a new cohort.
