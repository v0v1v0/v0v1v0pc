XYZtoRGB R Documentation

convert from XYZ coordinates to RGB

Description

XYZtoRGB convert from XYZ coordinates to RGB.

Usage

XYZtoRGB(xc, yc, zc, ColorSystem = c(0.67, 0.33, 0.21, 0.71,
                 0.14, 0.08, 0.31, 0.316))

Arguments

xc

XYZ X coordinates
yc

XYZ Y coordinates
zc

XYZ Z coordinates
ColorSystem

RGB Color System data

Value

RGB coordinates
