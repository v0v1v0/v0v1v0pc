sCD14 R Documentation

Microbiome and sCD14 inflammation parameter

Description

A dataset containing the number of counts of 60 different genera in a group of 151 samples (including HIV - infected and non - infected patients). The data.frame is composed by 60 genera and a numeric variable

Format

The data.frame is composed by 60 genera and a variable

genera

The first 60 columns, from g_Prevotella until o_NB1-n_g_unclassified referred to different genera.

sCD14

a numeric variable with the value of the inflammation parameter sCD14 for each sample.
