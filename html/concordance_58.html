hs5_sitc1 R Documentation

HS5-SITC1 Concordance

Description

A dataset containing concordances between HS5 and SITC1 classification.

Usage

hs5_sitc1

Format

A data frame with 5381 rows and 8 variables:

HS5_6d

6-digit HS5 Code

HS5_4d

4-digit HS5 Code

HS5_2d

2-digit HS5 Code

SITC1_5d

5-digit SITC1 Code

SITC1_4d

4-digit SITC1 Code

SITC1_3d

3-digit SITC1 Code

SITC1_2d

2-digit SITC1 Code

SITC1_1d

1-digit SITC1 Code

Source

https://unstats.un.org/unsd/trade/classifications/correspondence-tables.asp
