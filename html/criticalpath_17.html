sch_critical_relations R Documentation

Critical Relations

Description

Return a tibble with critical relations of a schedule in topological order.

Usage

sch_critical_relations(sch, order = "topological")

Arguments

sch

A schedule object.
order

Indicates the order of relations:

  • "topological": The relations tibble is in topological order.

  • "insert": The relations tibble is in insert order.

Value

A tibble with critical relations.
