Detailed report

Usage

reports_detailed(start, end, extra_args = list())

Arguments

end

End time
extra_args

Extra arguments to be passed to the API. Example: extra_args = list(rounding = TRUE).

Value

A data frame with detailed time entries for the specified time period.
