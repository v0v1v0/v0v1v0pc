summary.CPMCGLM R Documentation

Short summary of a CPMCGLM object

Description

The function provides the summary of a CPMCGLM correction of pvalue.

Usage

## S3 method for class 'CPMCGLM'
summary(object,...)

Arguments

object

an object inheriting from classes CPMCGLM.

...

other parameters.

Author(s)

J.Riou, A.Diakite, and B.Liquet

See Also

CPMCGLM,print.CPMCGLM

Examples

## Not run: 
# load data
data(data_sim)

#Linear Gaussian Model
fit1 <- CPMCGLM(formula= Weight~Age+as.factor(Sport)+Desease+Height,
family="gaussian",link="identity",data=data_sim,varcod="Age",N=1000,
boxcox=c(0,1,2,3))
### summary fit1
summary(fit1)

## End(Not run)
