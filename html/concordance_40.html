hs3_sitc4 R Documentation

HS3-SITC4 Concordance

Description

A dataset containing concordances between HS3 and SITC4 classification.

Usage

hs3_sitc4

Format

A data frame with 5050 rows and 8 variables:

HS3_6d

6-digit HS3 Code

HS3_4d

4-digit HS3 Code

HS3_2d

2-digit HS3 Code

SITC4_5d

5-digit SITC4 Code

SITC4_4d

4-digit SITC4 Code

SITC4_3d

3-digit SITC4 Code

SITC4_2d

2-digit SITC4 Code

SITC4_1d

1-digit SITC4 Code

Source

https://wits.worldbank.org/product_concordance.html
