chk_vector R Documentation

Check Vector

Description

Checks if is a vector using

(is.atomic(x) && !is.matrix(x) && !is.array(x)) || is.list(x)

Usage

chk_vector(x, x_name = NULL)

vld_vector(x)

Arguments

x

The object to check.
x_name

A string of the name of object x or NULL.

Details

is.vector(x) is not reliable because it returns TRUE only if the object is a vector with no attributes apart from names.

Value

The chk_ function throws an informative error if the test fails or returns the original object if successful so it can used in pipes.

The vld_ function returns a flag indicating whether the test was met.

Functions

  • vld_vector(): Validate Vector
