Laplace (i.e. add constant) smoothing

Description

Laplace (i.e. add constant) smoothing

Usage

laplace(freq, add = 0.5)

Arguments

freq

A numeric vector of term frequencies (integers).
add

The added value

Value

A numeric vector with the smoothed term proportions

Examples

laplace(c(0,0,1,1,1,2,2,2,3,3,4,7,10))
