obs.v.exp R Documentation

Function to plot a visualization of the observed versus expected oc-occurrences from an analysis stored in a coccurr object.

Description

Plot the observed number of co-occurrences versus the number expected from the probability analysis in a cooccur object.

Usage

Arguments
