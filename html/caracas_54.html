fraction_parts R Documentation

Get numerator and denominator of a fraction

Description

Get numerator and denominator of a fraction

Usage

fraction_parts(x)

numerator(x)

denominator(x)

Arguments

x

Fraction

Examples

if (has_sympy()) {
     x <- as_sym("a/b")
     frac <- fraction_parts(x)
     frac
     frac$numerator
     frac$denominator
 }
