vbz R Documentation

Sample files provided by VBZ

Description

Sample data of three distinct routes.

Usage

vbz

Format

list with data.frame elements

trip_seq

Sequence order of trips.

stop_seq

Sequence order of stops.

stop_name

Name of the stop.

trip_id

Id of trip

circulation_name

Name of circulation.

line_name

Name of line.

vehicle

Type of vehicle.

occupancy

Occupancy.

occ_category

Category of occupancy.

departure_time

Time of departure.

number_of_measurements

Number of measurements.

occ_cat_name

Occupancy category name

direction

Direction.

Source

vbz
