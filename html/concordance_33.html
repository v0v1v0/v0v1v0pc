hs3_hs0 R Documentation

HS3-HS0 Concordance

Description

A dataset containing concordances between HS3 and HS0 classification.

Usage

hs3_hs0

Format

A data frame with 5053 rows and 6 variables:

HS3_6d

6-digit HS3 Code

HS3_4d

4-digit HS3 Code

HS3_2d

2-digit HS3 Code

HS0_6d

6-digit HS0 Code

HS0_4d

4-digit HS0 Code

HS0_2d

2-digit HS0 Code

Source

https://unstats.un.org/unsd/trade/classifications/correspondence-tables.asp
