PlotAuppervAlower R Documentation

Plot upper-versus-lower link angles

Description

High-level function for plotting upper-versus-lower link angles.

Usage

PlotAuppervAlower(community, 
                  main=CPS(community)$title,
                  xlab=~A[lower], 
                  ylab=~A[upper], 
                  xlim=c(-180, 180),
                  ylim=c(-180, 180), 
                  pch=19, 
                  ...)

Arguments

community

an object of class Community.
main

title of the plot
xlab

title of the x axis.
ylab

title of the y axis.
xlim

limits of the x axis.
ylim

limits of the y axis.
pch

plotting symbol.
...

other values to plot functions.

Author(s)

Lawrence Hudson
