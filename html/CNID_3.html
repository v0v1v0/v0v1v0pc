birth_date R Documentation

Get date of birth information from ID number

Description

Get date of birth information from ID number.

Usage

birth_date(id)

Arguments

id

A vector of ID numbers.

Value

Date of birth vector obtained from ID numbers.

Examples

id = c(
"652801197305161555",
"130206202202291545", 
"110101841125178"
)
birth_date(id)
