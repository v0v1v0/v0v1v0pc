path_fill R Documentation

Fill In Path

Description

Calculate the coordinates of all points inside of a path.

Usage

path_fill(mat)

Arguments

mat

Nx2 matrix of (row, column) path coordinates

Value

Nx2 matrix of (row, column) coordinates

See Also

Other path-fitting functions: path_arc(), path_bezier(), path_circle(), path_ellipse(), path_fn(), path_intersection(), path_lerp(), path_ray(), path_rect(), path_shape()

Examples

c0 <- path_circle(c(10,10), r=5)
path_fill(c0)
