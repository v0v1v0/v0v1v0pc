empty.server R Documentation

Empty server function

Description

For use in simple shiny apps not requiring server functions

Usage

empty.server(input, output, session)

Arguments

input

the input object
output

the output object
session

the session object

Value

Empty server function

Examples

if(interactive()){
  library(shiny)
  library(card.pro)
  ui = fluidPage("Obi Obianom")
  shinyApp(ui = ui, server = empty.server)
}
