chk_all R Documentation

Check All

Description

Checks all elements using

all(vapply(x, chk_fun, TRUE, ...))

Usage

chk_all(x, chk_fun, ..., x_name = NULL)

vld_all(x, vld_fun, ...)

Arguments

x

The object to check.
chk_fun

A chk_ function.
...

Additional arguments.
x_name

A string of the name of object x or NULL.
vld_fun

A vld_ function.

Value

The chk_ function throws an informative error if the test fails or returns the original object if successful so it can used in pipes.

The vld_ function returns a flag indicating whether the test was met.

Functions

  • vld_all(): Validate All

See Also

Other chk_alls: chk_all_equal(), chk_all_equivalent(), chk_all_identical()

Examples

# chk_all
chk_all(TRUE, chk_lgl)
# FIXME try(chk_all(1, chk_lgl))
chk_all(c(TRUE, NA), chk_lgl)
# vld_all
vld_all(c(TRUE, NA), vld_lgl)
