n_borda R Documentation

Normalise using Borda scores

Description

Calculates Borda scores as rank(x) - 1.

Usage

n_borda(x, ties.method = "min")

Arguments

x

A numeric vector
ties.method

This argument is passed to base::rank() - see there for details.

Value

Numeric vector

Examples

x <- runif(20)
n_borda(x)
