genOrthogonal R Documentation

Generate An Orthogonal Matrix

Description

Generate an orthogonal matrix with given dimension.

Usage

genOrthogonal(dim)

Arguments

dim

integer. Dimension of the orthogonal matrix.

Value

An orthogonal matrix with dimension dim.

Examples

set.seed(12345)
Q = genOrthogonal(3)
print(Q)
A = Q 
print(A)
