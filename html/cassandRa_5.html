Compute Basic Confidence Intervals

Usage

ComputeCI(df)

Arguments

Value

a dataframe detailing confidence intervals at each tested sample size

Examples

 data(Safariland, package = 'bipartite')
 X<-RarefyNetwork(Safariland, n_per_level = 100)
