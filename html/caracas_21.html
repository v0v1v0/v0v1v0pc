collect R Documentation

Collects common powers of a term in an expression

Description

Collects common powers of a term in an expression

Usage

collect(x, a)

Arguments

x, a

A caracas_symbol

Examples

if (has_sympy()){
  def_sym(x, y, z)
  expr = x*y + x - 3 + 2*x**2 - z*x**2 + x**3
  collect(expr, x)
}
