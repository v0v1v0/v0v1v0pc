get_variant_renderings R Documentation

Render a Variant

Description

Get details about renderings (i.e. render history) or execute a variant on demand

Usage

get_variant_renderings(variant)

variant_render(variant)

Arguments

variant

An R6 Variant object. As returned by get_variant() or get_variant_default()

Details

  • get_variant_renderings() returns all renderings / content for a particular variant. Returns a tibble

  • variant_render() executes a variant on demand. Returns a VariantTask object
