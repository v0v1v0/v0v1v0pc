fastMedian R Documentation

Fast implementation of the median

Description

Compute the sample median with a fast C++ implementation.

Usage

fastMedian(x)

Arguments

x

a numeric vector.

Value

The sample median.

Note

Functionality for removing observations with missing values is currently not implemented.

Author(s)

Andreas Alfons

See Also

fastMAD, median

Examples

set.seed(1234)  # for reproducibility
x <- rnorm(100)
fastMedian(x)
