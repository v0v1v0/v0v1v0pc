chk_count R Documentation

Check Count

Description

Checks if non-negative whole number using

vld_whole_number(x) && x >= 0

Usage

chk_count(x, x_name = NULL)

vld_count(x)

Arguments

x

The object to check.
x_name

A string of the name of object x or NULL.

Value

The chk_ function throws an informative error if the test fails or returns the original object if successful so it can used in pipes.

The vld_ function returns a flag indicating whether the test was met.

Functions

  • vld_count(): Validate Count

See Also

Other chk_typeof: chk_character(), chk_character_or_factor(), chk_double(), chk_environment(), chk_factor(), chk_integer(), chk_list(), chk_logical()

Examples

# chk_count
chk_count(1)
try(chk_count(1.5))
# vld_count
vld_count(1)
vld_count(0L)
vld_count(-1)
vld_count(0.5)
