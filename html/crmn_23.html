Usage

## S4 replacement method for signature 'ExpressionSet,matrix'

mexprs(object) <- value

Arguments

object

an ExpressionSet or matrix
value

the value to assign

Details

Set the expression data in an ExpressionSet or just return the given matrix

Value

the expression data

Author(s)

Examples
