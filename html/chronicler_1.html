Pipe a chronicle object to a decorated function.

Description

Pipe a chronicle object to a decorated function.

Usage

.c %>=% .f

Arguments

.c

A value returned by record.
.f

A chronicle function to apply to the returning value of .c.

Value

A chronicle object.

Examples

r_sqrt <- record(sqrt)
r_exp <- record(exp)
3 |> r_sqrt() %>=% r_exp()
