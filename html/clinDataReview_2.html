addFacetPanel R Documentation

Add facet-panel to single plotly plot.

Description

Add facet-panel to single plotly plot.

Usage

addFacetPanel(
  pl,
  panelLab,
  panelWidth = 20,
  fontSize = 15,
  side = c("top", "right")
)

Arguments

pl

plotly object.
panelLab

text to be shown in the facet panel
panelWidth

thickness of the panel in pixels.
fontSize

fontsize of facetText
side

the side of the plot to show the panel (currently only right panels are implemented.)

Details

plot title clipping.

In case side = 'top', the plot title (eg. layout(title = "title")) will clip with the top panel.

Resolve this with the following configurations: (once all the subplots have already been combined) layout( # place the title at absolute top of the page title = list(text = "title", yref = "container", y = 1)) # If font size = 15 roughly equal to 20 pixels. margin = list(t = panelWidth + heightTitleTextInPixels)

Value

plotly object with the facet panel added.
