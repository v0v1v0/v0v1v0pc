ones R Documentation

Generates a matrix of ones

Description

ones generates a matrix of ones.

Usage

ones(n, p = n)

Arguments

n

number of rows

p

number of columns

Details

if only n is specified, the resulting matrix will be square.

Value

a matrix filled with ones.

Author(s)

Jan Graffelman (jan.graffelman@upc.edu)

See Also

matrix

Examples

Id <- ones(3)
print(Id)
