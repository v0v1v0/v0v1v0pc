C2 R Documentation

A covariance matrix

Description

Marginal covariance matrix C2 used for simulations in paper http://arxiv.org/abs/1505.02023

Usage

C2

Format

An object of class matrix with 7 rows and 7 columns.

Details

This is a 7x7 real-valued covariance matrix.
