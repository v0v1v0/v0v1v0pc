data_glass R Documentation

The glass dataset

Description

A dataset containing spectra with d = 750 wavelengths collected on n = 180 archeological glass samples.

Usage

data("data_glass")

Format

A data frame with 180 observations of 750 wavelengths.

Source

Lemberge, P., De Raedt, I., Janssens, K.H., Wei, F., and Van Espen, P.J. (2000). Quantitative Z-analysis of 16th-17th century archaeological glass vessels using PLS regression of EPXMA and \mu-XRF data. Journal of Chemometrics, 14, 751â€“763.
