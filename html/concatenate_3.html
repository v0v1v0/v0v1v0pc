infix R Documentation

Binary Infix Concatenation

Description

%+%:

binary infix operator for strings.

% + %:

like %+% but with a space between its inputs.

%,%, %or%, %and%:

infix versions of cc, cc_or, cc_and.

Usage

x %+% y

x % + % y

x %,% y

x %or% y

x %and% y

Arguments

x, y

Character vectors.

Examples

v <- "important value"
v %+% "!"

message("Two" % + % "words")
