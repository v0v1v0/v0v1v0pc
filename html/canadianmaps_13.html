label_prov R Documentation

Province labels

Description

Adds text labels in the center of each province with light gray bubble.

Usage

label_prov(
  data = PROV,
  label = "PT",
  colour = "grey20",
  size = 3,
  outline = NA,
  alpha = 0.7
)

Arguments

data

a data set with geometry variable
label

the label variable
colour

label color
size

label size
outline

outline color
alpha

transparency

Value

Provincial map labels
