chk_all_identical R Documentation

Check All Identical

Description

Checks all elements in x identical using

length(x) < 2L || all(vapply(x, vld_identical, TRUE, y = x[[1]]))

Pass: c(1, 1, 1), list(1, 1)

Fail: c(1, 1.0000001), list(1, NA)

Usage

chk_all_identical(x, x_name = NULL)

vld_all_identical(x)

Arguments

x

The object to check.
x_name

A string of the name of object x or NULL.

Value

The chk_ function throws an informative error if the test fails or returns the original object if successful so it can used in pipes.

The vld_ function returns a flag indicating whether the test was met.

Functions

  • vld_all_identical(): Validate All Identical

See Also

Other chk_alls: chk_all(), chk_all_equal(), chk_all_equivalent()

Examples

# chk_all_identical
chk_all_identical(c(1, 1))
try(chk_all_identical(c(1, 1.1)))
# vld_all_identical
vld_all_identical(c(1, 1))
