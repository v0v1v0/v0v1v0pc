plotModelCCC

Description

Creates a plot of a model implied category characteristic curve. Note this is not for use with rout files. See the generic function plotRout for plotting rout files.

Usage

plotModelCCC(item, range = c(-6, 6), by = 0.1, plotZero)

Arguments

item

Item parameters for a single item.
range

Lower and upper bounds to plot over (defaults to c(-6, 6).
by

Increment to the sequence along 'rangeâ€œ.
plotZero

Should the zero category be plotted? Defaults to FALSE when item is dichotomous and TRUE otherwise.

Value

A ggplot2 object.

Examples

myItem <- matrix(
  c(
    0, 0, 0, 1,
    1, 1, 0, 1
  ),
  ncol = 4, byrow=TRUE
)
myPlot <- plotModelCCC(myItem)
