ZIP Distribution R Documentation

COM-Poisson Distribution

Description

Functions for the COM-Poisson distribution.

Usage

dzip(x, lambda, p, log = FALSE)

rzip(n, lambda, p)

pzip(x, lambda, p)

qzip(q, lambda, p, log.p = FALSE)

ezip(lambda, p)

vzip(lambda, p)

Arguments

x

vector of quantiles.
lambda

rate parameter.
p

zero-inflation probability parameter.
log

logical; if TRUE, probabilities are returned on log-scale.
n

number of observations.
q

vector of probabilities.
log.p

logical; if TRUE, probabilities p are given as \log(p).

Value

dzip

density,

pzip

cumulative probability,

qzip

quantiles,

rzip

generate random variates,

ezip

expected value,

vzip

variance,
