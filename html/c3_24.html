Description

Use RColorBrewer palettes

Usage

RColorBrewer(c3, pal = "Spectral")

## S3 method for class 'c3'
RColorBrewer(c3, pal = "Spectral")

Arguments

c3

c3 htmlwidget object

Value

c3

Examples

data.frame(a = c(1,2,3,2), b = c(2,4,1,5), c = c(5,3,4,1)) %>%
