reset R Documentation

Reset Console Style

Description

Turns off all text attributes and colors in the terminal.

Usage

reset()

Value

NULL

See Also

Other style functions: attr_off(), attr_on(), bg_off(), bg_on(), color_off(), color_pair(), fg_off(), fg_on(), make_bg(), make_fg(), make_style(), style()

Examples

attr_on("ul")
fg_on("red")
bg_on(c(10,60,205))
cat("Hello World!\n")

reset()
cat("Hello World!\n")
