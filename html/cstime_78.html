Last ISO yearweek (character) in ISO year

Usage

## Default S3 method:
isoyear_to_last_isoyearweek_c(x)

## S3 method for class 'character'
isoyear_to_last_isoyearweek_c(x)

## S3 method for class 'numeric'
isoyear_to_last_isoyearweek_c(x)

Arguments

x

ISO year, e.g. 2020

Value

ISO year-week in character, of the last ISO year
