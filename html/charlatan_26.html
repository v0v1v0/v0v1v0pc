ch_job R Documentation

Create fake jobs

Description

Create fake jobs

Usage

ch_job(n = 1, locale = NULL)

Arguments

n

(integer) number of things to get, any non-negative integer
locale

(character) the locale to use. Run JobProvider$new()$allowed_locales() for locales supported (default: en_US)

See Also

JobProvider

ch_job()
ch_job(10)
