prepare_plot_data R Documentation

Prepare data for plotting

Description

Prepare data for plotting

Usage

prepare_plot_data(
  multi.dat,
  querysnpid,
  query_trait_names,
  thresh_Ha = 0.5,
  thresh_Hc = 0.5,
  hmp = FALSE,
  cophe.plot = TRUE
)

Arguments

multi.dat

multi trait cophescan results returned from cophe.multitrait or multitrait.simplify
querysnpid

query variant
query_trait_names

vector of names of the query traits
thresh_Ha

Ha threshold to be displayed
thresh_Hc

Hc threshold to be displayed
hmp

return for heatmap
cophe.plot

default: TRUE, return for cophe_plot

Value

plot list

See Also

cophe_plot, cophe.susie, cophe.multitrait, multitrait.simplify default NULL
