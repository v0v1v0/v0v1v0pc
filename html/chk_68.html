chk_not_subset R Documentation

Check Not Subset

Description

Checks if not all values in values using

!any(x %in% values) || !length(x)

Usage

chk_not_subset(x, values, x_name = NULL)

Arguments

x

The object to check.
values

A vector of the permitted values.
x_name

A string of the name of object x or NULL.

Value

The chk_ function throws an informative error if the test fails or returns the original object if successful so it can used in pipes.

The vld_ function returns a flag indicating whether the test was met.

See Also

Other chk_set: chk_join(), chk_orderset(), chk_superset(), vld_not_subset(), vld_orderset()

Examples

# chk_not_subset
chk_not_subset(11, 1:10)
try(chk_not_subset(1, 1:10))
