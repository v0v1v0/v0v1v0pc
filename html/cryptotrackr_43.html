Description

coingecko_vs_currencies

Usage

coingecko_vs_currencies(timeout_seconds = 60)

Arguments

timeout_seconds

seconds until the query times out. Default is 60.

Value

returns a character vector containing all supported currencies on Coingecko.

Examples
