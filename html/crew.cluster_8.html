Method terminate()

Terminate one or more SGE jobs.

Usage
crew_class_monitor_sge$terminate(jobs = NULL, all = FALSE)
Arguments
jobs

Character vector of job names or job IDs to terminate. Ignored if all is set to TRUE.

all

Logical of length 1, whether to terminate all the jobs under your user name. This terminates ALL your SGE jobs, regardless of whether crew.cluster launched them, so use with caution!

Returns

NULL (invisibly).
