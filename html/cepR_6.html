busca_multi R Documentation

Busca por CEP

Description

Busca usando multiplos CEP (search for information using multiple postal code).

Usage

busca_multi(lista_ceps = c("01001000", "01001001"), token = NULL)

Arguments

lista_ceps

lista de CEPs (postal code list).
token

Token de autorização. Veja <http://cepaberto.com/users/register>.
