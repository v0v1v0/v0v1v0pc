pair.profile R Documentation

Function to produce a visualization of species contributions to co-occurrence patterns.

Description

Plots a bar plot for visualizing the associations of each individual species from a cooccur object.

Usage

pair.profile(mod)

Arguments

mod

Object of class cooccur.

Author(s)

Daniel M Griffith
