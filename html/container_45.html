clone R Documentation

Clone an Object

Description

Creates a copy of the object.

Usage

clone(x)

## S3 method for class 'Container'
clone(x)

## S3 method for class 'dict.table'
clone(x)

Arguments

x

any R object.

Value

A copy of the object.

Examples


co = container(1, 2, 3)
co2 = clone(co)
co == co2

d = dict.table(a = 1:2, b = 3:4)
d2 = clone(d)
ref_clear(d)
print(d2)
