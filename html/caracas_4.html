add_prefix R Documentation

Add prefix to each element of matrix

Description

Add prefix to each element of matrix

Usage

add_prefix(x, prefix = "")

Arguments

x

Numeric or symbolic matrix
prefix

A character vector

Examples

if (has_sympy()) {
  X <- matrix_sym(2, 3)
  X
  add_prefix(X, "e")
  
  X <- matrix(1:6, 3, 2)
  X
  add_prefix(X, "e")
}
