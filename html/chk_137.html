chk_identical R Documentation

Check Identical

Description

Checks if is identical to y using

identical(x, y)

Usage

chk_identical(x, y, x_name = NULL)

vld_identical(x, y)

Arguments

x

The object to check.
y

An object to check against.
x_name

A string of the name of object x or NULL.

Value

The chk_ function throws an informative error if the test fails or returns the original object if successful so it can used in pipes.

The vld_ function returns a flag indicating whether the test was met.

Functions

  • vld_identical(): Validate Identical

See Also

Other chk_equals: chk_equal(), chk_equivalent()

Examples

# chk_identical
chk_identical(1, 1)
try(chk_identical(1, 1L))
chk_identical(c(1, 1), c(1, 1))
try(chk_identical(1, c(1, 1)))
vld_identical(1, 1)
