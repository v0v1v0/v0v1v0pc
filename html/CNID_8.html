cnid_info R Documentation

Get full information from ID number

Description

Get full information from ID number.

Usage

cnid_info(id)

Arguments

id

A vector of ID numbers.

Value

A list about date of birth, age, gender, etc. obtained from ID number.

Examples

id = c(
"652801197305161555",
"130206202202291545", 
"110101841125178"
)
cnid_info(id)
