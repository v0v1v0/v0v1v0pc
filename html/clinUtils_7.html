colorPaletteNRIND R Documentation

Color palette for a standard CDISC Normal/Reference Range Indicator.

Description

Color palette for a standard CDISC Normal/Reference Range Indicator.

Usage

colorPaletteNRIND

Format

A named character vector with color for typical Normal Reference Range Indicator variable:

  • "LOW": orange

  • "NORMAL": green4

  • "HIGH": orange

  • "ABNORMAL": red

  • "UNKNOWN" or 'NA': grey

  • "NA": grey
