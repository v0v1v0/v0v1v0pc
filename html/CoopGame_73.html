drawPublicHelpValue R Documentation

Draw Public Help value Theta for 3 or 4 players

Description

drawPublicHelpValue draws the (normalized) Public Help value Theta for 3 or 4 players.

Usage

drawPublicHelpValue(
  v,
  holdOn = FALSE,
  colour = NA,
  label = TRUE,
  name = "Normalized Public Help Value"
)

Arguments

v

Numeric vector of length 2^n - 1 representing the values of the coalitions of a TU game with n players
holdOn

draws in a existing plot
colour

draws the geometric object (i.e. point or convex polyhedron) with this colour, all colour names can be seen with "colors()"
label

activates the labels for the figure
name

set a name for the label

Value

None.
