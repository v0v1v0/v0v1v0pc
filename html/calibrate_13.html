rad2degree R Documentation

Convert radians to degrees.

Description

rad2degree converts radians to degrees.

Usage

rad2degree(x)

Arguments

x

an angle in radians

Value

the angle with the positive x-axis in degrees.

Author(s)

Jan Graffelman (jan.graffelman@upc.edu)

Examples

   x <- pi/2
   a <- rad2degree(x)
   cat("angle is",a,"degrees\n")
