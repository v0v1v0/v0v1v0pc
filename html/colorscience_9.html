CCT2XYZ R Documentation

Convert CCT to XYZ

Description

CCT2XYZ Converts correlated color temperature (CCT) to CIE tristimulus XYZ.

Usage

CCT2XYZ(CCTmatrix)

Arguments

CCTmatrix

CCT values

Value

CIE tristimulus XYZ

Author(s)

Jose Gama

Source

Bruce Justin Lindbloom, 2013 Color Calculator http://www.brucelindbloom.com/

References
