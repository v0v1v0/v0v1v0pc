refresh_covid19swiss R Documentation

Description

aily summary of the Coronavirus (COVID-19) pandemic cases in Switzerland by Canton

Usage

refresh_covid19swiss()

Value

A data.frame object

Source

Specialist Unit for Open Government Data Canton of Zurich website

Examples


# update the data
covid19_swiss <- refresh_covid19swiss()
