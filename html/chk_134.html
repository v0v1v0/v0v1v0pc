chk_function R Documentation

Check Function

Description

Checks if is a function using

is.function(x) && (is.null(formals) || length(formals(x)) == formals)

Usage

chk_function(x, formals = NULL, x_name = NULL)

vld_function(x, formals = NULL)

Arguments

x

The object to check.
formals

A count of the number of formal arguments.
x_name

A string of the name of object x or NULL.

Value

The chk_ function throws an informative error if the test fails or returns the original object if successful so it can used in pipes.

The vld_ function returns a flag indicating whether the test was met.

Functions

  • vld_function(): Validate Function

Examples

# vld_function
