scale_xifti R Documentation

Scale CIFTI

Description

Scale CIFTI data. Similar to scale.

Usage

scale_xifti(xifti, center = TRUE, scale = TRUE)

Arguments

xifti

A "xifti" object.
center, scale

Arguments to scale.

Value

The input "xifti" with scaled columns.
