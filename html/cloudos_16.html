cohort-class R Documentation

cohort class

Description

This class creates a cohort object, which holds the information related to a cohort: cohort ID, name, description, query, table columns. This class is used in functions which carry out operations related to specific cohorts. A cohort class object can be created using constructor functions cb_create_cohort or cb_load_cohort.

Slots

id

cohort ID.

name

cohort name.

desc

cohort description.

phenoptype_filters

phenotypes displayed in the cohort overview.

query

applied query.

query_phenotype_ids

IDs of phenotypes used in the query.

columns

All the columns.

num_participants

number of participants in the cohort.

cb_version

chort browser version.
