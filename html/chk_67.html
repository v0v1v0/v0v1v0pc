chk_not_null R Documentation

Check not NULL

Description

Checks if not NULL using

!is.null(x)

Usage

chk_not_null(x, x_name = NULL)

vld_not_null(x)

Arguments

x

The object to check.
x_name

A string of the name of object x or NULL.

Value

The chk_ function throws an informative error if the test fails or returns the original object if successful so it can used in pipes.

The vld_ function returns a flag indicating whether the test was met.

Functions

  • vld_not_null(): Validate Not NULL

See Also

Other chk_nulls: chk_null()

Examples

# chk_not_null
try(chk_not_null(NULL))
chk_not_null(1)
# vld_not_null
vld_not_null(1)
vld_not_null(NULL)
