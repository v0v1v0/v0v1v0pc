Turn a psych::alpha() object into HTML tables.

Usage

knit_print.alpha(x, indent = "######", ...)

Arguments

x

a psych alpha object
indent

add # to this to make the headings in the components lower-level. defaults to beginning at h5
...

ignored

example("alpha", "psych")
