is_zoned_time R Documentation

Is x a zoned-time?

Description

This function determines if the input is a zoned-time object.

Usage

is_zoned_time(x)

Arguments

x

⁠[object]⁠

An object.

Value

TRUE if x inherits from "clock_zoned_time", otherwise FALSE.

Examples

is_zoned_time(1)
is_zoned_time(zoned_time_now("America/New_York"))
