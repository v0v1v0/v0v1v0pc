as_nested_list R Documentation

ARD as Nested List

Description


Convert ARDs to nested lists.

Usage

as_nested_list(x)

Arguments

x

(data.frame)
an ARD data frame of class 'card'

Value

a nested list

Examples

ard_continuous(mtcars, by = "cyl", variables = c("mpg", "hp")) |>
  as_nested_list()
