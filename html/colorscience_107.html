PhotoYCC2RGB R Documentation

Convert PhotoYCC to RGB

Description

PhotoYCC2RGB Converts PhotoYCC to RGB.

Usage

PhotoYCC2RGB(PhotoYCCmatrix)

Arguments

PhotoYCCmatrix

PhotoYCC coordinates

Value

RGB coordinates
