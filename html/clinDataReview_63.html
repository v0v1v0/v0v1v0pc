getPathHyperlink R Documentation

Get path ('href') property from hyperlink(s).

Description

Get path ('href') property from hyperlink(s).

Usage

getPathHyperlink(x)

Arguments

x

Character vector with hyperlink(s). If multiple, the hyperlinks should be separated by: ', '.

Value

Character vector of length x containing only the hyperlinks.
