hs2_hs0 R Documentation

HS2-HS0 Concordance

Description

A dataset containing concordances between HS2 and HS0 classification.

Usage

hs2_hs0

Format

A data frame with 5223 rows and 6 variables:

HS2_6d

6-digit HS2 Code

HS2_4d

4-digit HS2 Code

HS2_2d

2-digit HS2 Code

HS0_6d

6-digit HS0 Code

HS0_4d

4-digit HS0 Code

HS0_2d

2-digit HS0 Code

Source

https://unstats.un.org/unsd/trade/classifications/correspondence-tables.asp
