chk_false R Documentation

Check FALSE

Description

Check if FALSE using

is.logical(x) && length(x) == 1L && !anyNA(x) && !x

Usage

chk_false(x, x_name = NULL)

vld_false(x)

Arguments

x

The object to check.
x_name

A string of the name of object x or NULL.

Value

The chk_ function throws an informative error if the test fails or returns the original object if successful so it can used in pipes.

The vld_ function returns a flag indicating whether the test was met.

Functions

  • vld_false(): Validate FALSE

See Also

Other chk_logical: chk_flag(), chk_lgl(), chk_true()

Examples

# chk_false
chk_false(FALSE)
try(chk_false(0))
# vld_false
vld_false(TRUE)
vld_false(FALSE)
vld_false(NA)
vld_false(0)
vld_false(c(FALSE, FALSE))
