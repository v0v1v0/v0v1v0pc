distanceRealEuclidean R Documentation

Euclidean Distance

Description

The Euclidean distance for real vectors.

Usage

distanceRealEuclidean(x, y)

Arguments

x

first real vector
y

second real vector

Value

numeric distance value

d(x,y)

Examples

x <- runif(5)
y <- runif(5)
distanceRealEuclidean(x,y)
