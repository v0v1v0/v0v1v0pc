is_naive_time R Documentation

Is x a naive-time?

Description

This function determines if the input is a naive-time object.

Usage

is_naive_time(x)

Arguments

x

⁠[object]⁠

An object.

Value

TRUE if x inherits from "clock_naive_time", otherwise FALSE.

Examples

is_naive_time(1)
is_naive_time(as_naive_time(duration_days(1)))
