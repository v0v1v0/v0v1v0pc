REapt R Documentation

Reparto equitativo entre os apartamentos

Description

Calcula a cantidade a pagar do custo do ascensor por un apartamento en base á regra do Reparto Equitativo. Emprega como unidade de reparto os apartamentos.

Usage

REapt(custo, napt)

Arguments

custo

O custo total da instalación do ascensor
napt

O número de apartamentos que ten o edificio

Value

A cantidade que lle corresponde pagar a cada apartamento

Examples

REapt(140,7)
REapt(120,4)
