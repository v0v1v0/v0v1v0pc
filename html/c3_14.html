c3_scatter R Documentation

Scatter Plots

Description

For scatter plots options are defined in the 'c3' function. Options are limited to x, y and groups

Usage

c3_scatter(c3)

Arguments

c3

c3 htmlwidget object

Value

c3

Examples

  iris %>%
    c3(x = 'Sepal_Length',
       y = 'Sepal_Width',
       group = 'Species') %>%
    c3_scatter()
