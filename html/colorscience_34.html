CMY2CMYK R Documentation

Convert CMY coordinates to CMYK

Description

CMY2CMYK Converts CMY coordinates to CMYK.

Usage

CMY2CMYK(CMYmatrix)

Arguments

CMYmatrix

CMY coordinates

Value

CMYK coordinates
