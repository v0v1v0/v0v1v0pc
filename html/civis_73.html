databases_get_table_privilegesschema_name R Documentation

Show table privileges

Description

Show table privileges

Usage

databases_get_table_privilegesschema_name(id, schema_name, table_name)

Arguments

id

integer required. The ID of the database
schema_name

string required. The name of the schema
table_name

string required. The name of the table

Value

A list containing the following elements:

grantee

string, Name of the granted user or group
granteeType

string, User or group
privileges

array, Privileges that the grantee has on this resource
grantablePrivileges

array, Privileges that the grantee can grant to others for this resource
