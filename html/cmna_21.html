fibonacci R Documentation

Fibonacci numbers

Description

Return the n-th Fibonacci number

Usage

fibonacci(n)

Arguments

n

n

Details

This function is recursively implements the famous Fibonacci sequence. The function returns the nth member of the sequence.

Value

the sequence element
