Description

Plots the uniformity of a color palette

Usage

PlotUniformity(pal, color.space = "LAB")

Arguments

pal

A colorspace palette
color.space

the color space (see colorspace::color)

Value

a ggplot instance

Examples

data("OptimGreenRedLAB")
p <- PlotUniformity(OptimGreenRedLAB)
plot(p)
