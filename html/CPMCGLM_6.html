data_sim R Documentation

dataset for CPMCGLM package.

Description

This simulated dataset contains 100 subjects and 8 variables.

Usage

data(data_sim)

Format

A data frame with 100 observations on the following 8 variables.

Height

a numeric vector

Weight

a numeric vector

Age

a numeric vector

Smoke

a binary vector

Sport

a numeric vector

Desease

a numeric vector

Parameter

a binary vector

Stroke

a categorical vector

Examples

data(data_sim)
head(data_sim)
