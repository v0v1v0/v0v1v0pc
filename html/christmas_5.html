xmasflag R Documentation

Christmas flag.

Description

Christmas flag (2015 card). A flag with a Christmas message.

Usage

xmasflag(
  year = 2016,
  language = c("english", "spanish", "catalan"),
  seed = NULL
)

Arguments

year

Year to be printed. Default is 2016.
language

Language to be used in the card. One of c("english", "spanish", "catalan"). Default is "english".
seed

Seed for reproducibility of the card. Default is NULL (no seed).

Value

A Christmas card plot including a message in a flag.

Author(s)

Jose Barrera-Gomez.

Examples


xmasflag(year = 2020, language = "catalan")
