CatDynData R Documentation

Class Attribute of CatDyn Data Object

Description

To be used by plot.CatDynData() to examine raw data and CatDynFit() to fit models.

Usage

CatDynData(x, ...)

Arguments

x

A list object coming from as.CatDynData().

...

Not used.

Value

A class attribute.
