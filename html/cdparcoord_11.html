showCounts R Documentation

Show tuple counts for the most recent saved counting operation.

Description

Used with saveCounts TRUE in tupleFreqs etc. to recover the tuple counts.

Usage

    showCounts(nshow=NULL)

Arguments

nshow

Dataset to show.
