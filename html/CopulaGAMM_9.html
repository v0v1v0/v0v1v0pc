dfrk R Documentation

B3 bivariate Frank copula density

Description

Density at (u,v)

Usage

dfrk(u, v, cpar)

Arguments

u

vector of values in (0,1)
v

vector of values in (0,1)
cpar

copula parameter, cpar>0 or cpar<0

Value

out

Vector of densities

Author(s)

Pavel Krupskii

Examples

out = dfrk(0.3,0.5,2)
