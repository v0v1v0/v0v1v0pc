curvature-comp R Documentation

Curvature of Composition

Description

Determine whether a composition is non-decreasing or non-increasing in an index.

Usage

is_incr(object, idx)

is_decr(object, idx)

## S4 method for signature 'Atom'
is_incr(object, idx)

## S4 method for signature 'Atom'
is_decr(object, idx)

Arguments

object

A Atom object.
idx

An index into the atom.

Value

A logical value.

Examples

x <- Variable()
is_incr(log(x), 1)
is_incr(x^2, 1)
is_decr(min(x), 1)
is_decr(abs(x), 1)
