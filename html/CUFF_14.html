pal_CUFF R Documentation

Palette de couleurs

Description

A 10 color palette.

Usage

pal_CUFF(n = 10, pal = "CUFF")

Arguments

n

Integer indicating the number of color desired (1-10)
pal

The only value possible for now is CUFF

Value

Returns a vector of color.

Author(s)

Charles-Édouard Giguère

Examples

pal_CUFF(3)
