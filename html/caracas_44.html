drop_remainder R Documentation

Remove remainder term

Description

Remove remainder term

Usage

drop_remainder(x)

Arguments

x

Expression to remove remainder term from

See Also

taylor()

Examples

if (has_sympy()) {
  def_sym(x)
  f <- cos(x)
  ft_with_O <- taylor(f, x0 = 0, n = 4+1)
  ft_with_O
  ft_with_O %>% drop_remainder() %>% as_expr()
}
