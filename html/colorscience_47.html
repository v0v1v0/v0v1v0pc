Description

deltaECMC The color difference method of the Color Measurement Committee (the CMC) .

Usage

deltaECMC(Lab1, Lab2, L, C)

Arguments

Lab1

CIE Lab color sample
Lab2

CIE Lab color reference
L

parameter L
C

parameter C

Value

Delta E
