Coppe.cosenza S4 Class

Description

Coppe.cosenza S4 class represents the solution of the COPPE-Cosenza method. In order to do so, this S4 class contains the final evaluation of the options regarding the studied projects. It presents a data frame presenting the final evaluation of the options regarding each project. If an option does not satisfies projectÂ´s specific factors, the option is discarded (a veto operation), with the value NA. The result also presents relevant messages list, describing if some evaluation could not be performed due to entry failures or missing evaluations.
