print.CGGP R Documentation

Print CGGP object

Description

Default print as a list is bad since there's a lot of elements.

Usage

## S3 method for class 'CGGP'
print(x, ...)

Arguments

x

CGGP object
...

Passed to print

Value

String to be printed

Examples

SG = CGGPcreate(3,21)
print(SG)
f <- function(x) {x[1]+exp(x[2]) + log(x[3]+4)}
y <- apply(SG$design, 1, f)
SG <- CGGPfit(SG, y)
print(SG)
