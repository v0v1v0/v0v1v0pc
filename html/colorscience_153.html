whitepointsRGB R Documentation

Primaries for RGB color spaces

Description

whitepointsRGB is a table with primaries for RGB color spaces.

Usage

whitepointsRGB

Format

This data frame contains the following data:

xRed

Primary red x

yRed

Primary red y

xGreen

Primary green x

yGreen

Primary green y

xBlue

Primary blue x

yBlue

Primary blue y

whitepointilluminant

illuminant

gamma

gamma

description

Color space name

Jose Gama
