Groups_Drug R Documentation

Drug Groups

Description

Groups that this drug belongs to.

Usage

Groups_Drug

Format

a tibble with 2 variables:

group

one of the above values

drugbank-id

drugbank id

Details

Each drug may have one or more category.

Groups include:

  • approved

  • vet_approved

  • nutraceutical

  • illicit

  • withdrawn

  • investigational

  • experimental
