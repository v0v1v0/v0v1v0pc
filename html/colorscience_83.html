LMS2RGB R Documentation

Convert LMS coordinates to RGB

Description

LMS2RGB Converts LMS coordinates to RGB.

Usage

LMS2RGB(LMSmatrix)

Arguments

LMSmatrix

LMS coordinates

Value

RGB coordinates

Author(s)

Jose Gama

Source

References
