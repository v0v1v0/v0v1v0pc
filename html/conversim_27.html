preprocess_text R Documentation

This file contains core similarity calculation functions such as topic similarity, lexical similarity, semantic similarity, structural similarity, stylistic similarity, sentiment similarity, participant similarity, and timing similarity.

Description

Preprocess text for analysis

Usage

preprocess_text(text)

Arguments

text

A character string to be preprocessed

Details

This function preprocesses the input text by converting to lowercase, removing punctuation and digits, and trimming whitespace.

Value

A preprocessed character string

Examples

text <- "Hello, World! This is an example text (with 123 numbers)."
preprocess_text(text)
