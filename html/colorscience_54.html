dkl2rgb R Documentation

convert RGB to DKL

Description

dkl2rgb Converts DKL, spherical coords coordinates to sRGB. dklCart2rgb Converts DKL, cartesian coords coordinates to sRGB.

Usage

dkl2rgb(dklMatrix, conversionMatrix = NA)

Arguments

dklMatrix

DKL coordinates
conversionMatrix

conversion matrix

Value

RGB coordinates
