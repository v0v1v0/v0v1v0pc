xmashealth R Documentation

Christmas risks.

Description

Christmas risks (2011 card). Some epidemiological jokes about potential effects of Christmas on health.

Usage

xmashealth(year = 2012, seed = NULL)

Arguments

year

Year to be printed. Default is 2012.
seed

Seed for reproducibility of the card. Default is NULL (no seed).

Value

A Christmas card plot including boxplots and OR estimates.

Author(s)

Jose Barrera-Gomez.

Examples


xmashealth()
