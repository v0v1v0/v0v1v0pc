draw_path R Documentation

Draw Path

Description

Draws text at each supplied coodinate.

Usage

draw_path(coord, text = "x", ...)

Arguments

coord

matrix or list containing (row, col) coordinates.
text

character value drawn at coordinate
...

parameters that are passed to style(), including the foreground color fg, background color bg, and attribute attr

Value

NULL

See Also

Other drawing functions: box_at(), draw_arc(), draw_bezier(), draw_circle(), draw_ellipse(), draw_fn(), draw_lerp(), draw_ray(), draw_rect(), draw_shape(), fill_circle(), fill_ellipse(), fill_rect(), fill_shape(), grid_at(), grid_mat(), hline_at(), hline(), vline_at(), vline()

Examples

c <- path_circle(yx = c(5,5), r=3)
draw_path(c, text="0")
