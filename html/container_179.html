rev R Documentation

Reverse Elements

Description

rev provides a reversed version of its argument.

Usage

ref_rev(x)

## S3 method for class 'Deque'
ref_rev(x)

## S3 method for class 'Deque'
rev(x)

Arguments

x

Deque object

Details

rev uses copy semantics while ref_rev works by reference, that is, it reverse all elements in place.

Value

For Deque, an object of class Deque

See Also

base::rev()

Examples


d = deque(a = 1, b = 2, 3)
rev(d)
print(d)
ref_rev(d)
print(d)
