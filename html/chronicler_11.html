pick R Documentation

Retrieve an element from a chronicle object.

Description

Retrieve an element from a chronicle object.

Usage

pick(.c, .e)

Arguments

.c

A chronicle object.
.e

Element of interest to retrieve, one of "value" or "log_df".

Value

The 'value' or 'log_df' element of the chronicle object .c.

Examples

r_sqrt <- record(sqrt)
r_exp <- record(exp)
3 |> r_sqrt() %>=% r_exp() |> pick("value")
