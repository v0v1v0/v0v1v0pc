pasture R Documentation

CDL pasture classes

Description

An array of CDL enumerations that contain pasture. The pasture enumeration contains:

  • 37 - Other Hay/Non Alfalfa

  • 38 - Camelina

  • 39 - Buckwheat

  • 62 - Pasture/Grass

  • 171 - Grassland Herbaceous

Usage

pasture

Format

An object of class numeric of length 5.
