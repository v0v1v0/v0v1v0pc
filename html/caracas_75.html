matrify R Documentation

Creates matrix from array symbol

Description

Creates matrix from array symbol

Usage

matrify(x)

Arguments

x

Array symbol to convert to matrix

Examples

if (has_sympy()) {
  x <- symbol("x")
  y <- symbol("y")
  f <- 3*x^2 + x*y^2
  matrify(f)
  h <- der2(f, list(x, y))
  h
  dim(h)
  H <- matrify(h)
  H
  dim(H)
}
