sitc4_sitc2 R Documentation

SITC4-SITC2 Concordance

Description

A dataset containing concordances between SITC4 and SITC2 classification.

Usage

sitc4_sitc2

Format

A data frame with 3271 rows and 10 variables:

SITC4_5d

5-digit SITC4 Code

SITC4_4d

4-digit SITC4 Code

SITC4_3d

3-digit SITC4 Code

SITC4_2d

2-digit SITC4 Code

SITC4_1d

1-digit SITC4 Code

SITC2_5d

5-digit SITC2 Code

SITC2_4d

4-digit SITC2 Code

SITC2_3d

3-digit SITC2 Code

SITC2_2d

2-digit SITC2 Code

SITC2_1d

1-digit SITC2 Code

Source

https://unstats.un.org/unsd/trade/classifications/correspondence-tables.asp
