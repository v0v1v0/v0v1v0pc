Hue.2.RGB R Documentation

Convert Hue to RGB

Description

Hue.2.RGB Converts Hue to RGB for HSL conversion.

Usage

Hue.2.RGB(v1, v2, vH)

Arguments

v1

value 1
v2

value 2
vH

value of hue

Value

RGB coordinates
