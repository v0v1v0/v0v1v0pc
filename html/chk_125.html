chk_double R Documentation

Check Double

Description

Checks if double using

is.double(x)

Usage

chk_double(x, x_name = NULL)

vld_double(x)

Arguments

x

The object to check.
x_name

A string of the name of object x or NULL.

Value

The chk_ function throws an informative error if the test fails or returns the original object if successful so it can used in pipes.

The vld_ function returns a flag indicating whether the test was met.

Functions

  • vld_double(): Validate Double

See Also

Other chk_typeof: chk_character(), chk_character_or_factor(), chk_count(), chk_environment(), chk_factor(), chk_integer(), chk_list(), chk_logical()

Examples

chk_double(1)

# vld_double
